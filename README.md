# InfoSetu Landing Page

Modern, fully responsive, animated landing page for the InfoSetu blog / YouTube brand.

## Tech
- Next.js
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide React

## Run locally
```bash
npm install
npm run dev
```

## Notes
- `public/logo.png` uses your uploaded logo.
- This is the landing page only. Articles, categories, and contact pages can be added later.
